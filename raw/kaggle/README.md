di repositori GitHub. Dataset lengkap dapat diakses melalui Google Drive
pada tautan berikut:

https://drive.google.com/file/d/1gv3Db92zRAK5fwU-o4JYO8vr3oYMepYJ/view

tanpa proses preprocessing atau cleaning.
