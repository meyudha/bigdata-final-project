Folder datalake digunakan sebagai penyimpanan terpusat untuk data mentah
(raw data) yang berasal dari berbagai sumber.

Data yang disimpan pada datalake masih dalam kondisi asli dan belum
melalui proses pembersihan, transformasi, atau integrasi.

Sumber data yang disimpan:
1. Data Kaggle (format CSV)
2. Data Hugging Face (format Parquet)

Datalake berfungsi sebagai penghubung antara proses extract
dan pipeline ETL maupun ELT.
